# gfycat_telegram_bot

Telegram bot for inline gif queries

## Installation

1. Download from this repo's releases.
2. Setup TELEGRAM_TOKEN environmental variable

## Usage


    $ java -jar gfycat_telegram_bot-0.1.0-standalone.jar
